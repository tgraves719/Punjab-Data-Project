# Prior-hand marginalia — 1911Q1 inserted leaf (doc[237])

An **unnumbered leaf bound in between printed p.43 (doc[236]) and printed p.44 (doc[238])** of the 1911Q1 catalogue carries the same prior-hand handwritten index documented for the 1910 quarters (see `../1910Q1/marginalia_p052.md` for provenance: an earlier, unidentified hand predating Prof. Davis's acquisition — confirmed with him July 2026). Same format: multi-part / serialized works keyed **`page.serial`** with a language tag.

Note the structural consequence: because this leaf is bound *inside* the quarter (not after it, as in 1910), the printed→pdf offset shifts by one for the final page — `manifest_1911Q1.json` covers pdf 195–236 at offset −193, and p.44 (pdf 238, offset −194) is rendered separately. High-res scan of this leaf: `marginalia_pdf237.png`.
